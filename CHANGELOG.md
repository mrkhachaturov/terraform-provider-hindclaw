## [0.1.0] - 2026-03-22

### Added

- Hindclaw resources for users, groups, channels, memberships, bank permissions, strategy scopes, and API keys.
- Hindsight resources for banks, bank configuration, mental models, directives, and webhooks.
- Data sources for resolved permissions, bank profiles, and bank listing.
- Acceptance-test scaffolding, generated documentation, runnable examples, and release automation.
- Terraform Registry and OpenTofu-ready release metadata, GitHub Actions workflows, and signed release support.
